달력

학습 도구 : Eclipse

사용 언어 : JAVA, JavaScript(jquery)

DB: Mysql

[개인학습 소개]

- 사용자가 입력한 년도와 월에 따라 동적으로 달력을 생성하는 기능

[구현 기능]

1. 사용자가 입력한 년도와 월을 받아와서 해당하는 달력을 표시

2. 입력된 년도와 월에 따라 해당 달의 첫 번째 날의 요일과 마지막 날짜를 계산

3. 이전 달과 다음 달로 이동할 수 있는 버튼을 제공하고, 클릭 시 해당 달력을 보여줌

4. 계산된 달의 날짜를 달력 형식으로 표시, 각 날짜는 요일에 따라 색상이 다르게 표시

5. 시작 요일 이전과 마지막 날짜 이후의 빈 공간을 채우기 위해 빈 칸을 추가

[개인학습 소감]

- 간단할 줄 알았지만 생각보다 사용자가 입력한 년 월에대한 날짜를 계산하는것에는 많은 처리가 필요하였다

- 해당월의 1일을 요일에 맞게 출력하기위해 빈칸 만들기

- for문안에서 if 조건으로 토,일 색깔처리와 개행을 위한 설정 등

- 동적으로 입력받은 정보를 출력하기위해 for문과 if문 while문등을 복합적으로 사용해보며 많은공부가 된것같다.

프로젝트를하며 무슨 공부를 하였는가?

데이터 바인딩의 흐름을 이해하기 위해 JSP마다 직관적으로 DB연결 후 데이터를 바인딩하여 CRUD를 구현하였고 다음 프로젝트에서는 SQL을 사용하여 데이터를 가공하고 조금더 디테일한 정보들을 출력해 보려고한다

`

open api를 사용하여 우편번호 검색도 해보고 Jquery를 사용하여 달력에서 직접 날짜를 가져오는 기능들을 추가해보면서 내가 모든것을 만들지않아도 이미 만들어져있는 기능들을 가져와서 활용하는 것 또한 굉장히 중요한 부분인것 같다고 느꼈다

간단할 줄 알았지만 생각보다 사용자가 입력한 년 월에대한 날짜를 계산하는것에는 많은 처리가 필요하였다 해당월의 1일을 요일에 맞게 출력하기위해 빈칸 만들기 for문안에서 if 조건으로 토,일 색깔처리와 개행을 위한 설정 등 동적으로 입력받은 정보를 출력하기위해 for문과 if문 while문등을 복합적으로 사용해보며 많은공부가 된것같다

기존에 만들었던 게시판에 추가적으로 로그인기능도 넣어보고 답변기능과 답변 수정기능을 조건을 주어 구현해보았다 조금씩 내가 생각한 기능들을 실제로 구현해보면서 뿌듯함을 느끼지만 다양한 기능을 구현하기위해 SQL문을 많이공부해야겠다는 생각이 들었다.

중복된 코드를 줄이기위해 include를 사용하였고 SQL문을 사용하여 CRUD(조회,입력,수정,삭제)를 구현해보며 기본적인 SQL문의 동작원리를 이해하였다.
